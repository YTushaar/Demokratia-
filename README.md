# Demokratia Youth Parliament 2026

A premium, responsive event website for the **Demokratia Youth Parliament**. Built with Next.js 15+ and Tailwind CSS, this platform features a modern design, interactive components, and a seamless registration experience.

## 🚀 Live Demo
[Live Website](https://demokratia-rust.vercel.app/)

## ✨ Features

- **Premium Design**: Sleek dark mode aesthetics with Royal Blue and Gold accents.
- **Interactive Sections**:
  - **Hero**: Dynamic entrance with a custom logo glow.
  - **Event Details**: Clickable location and calendar integration.
  - **Committees**: Interactive cards for AIPPM, Lok Sabha, and National Press.
  - **Executive Board**: Professional team showcase with hover effects.
  - **Chief Guests**: Animated marquee displaying distinguished speakers.
- **Embedded Registration**: Seamlessly integrated Fillout form for delegates.
- **Responsive**: Fully optimized for mobile, tablet, and desktop views.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Registration**: [Fillout](https://www.fillout.com/)
- **Fonts**: Space Grotesk & Instrument Serif (Google Fonts)

## 📦 Getting Started

### Prerequisites
- Node.js 18.x or later
- npm / pnpm / yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## 🌐 Deployment

The project is optimized for deployment on **Vercel**. 

1. Push your code to GitHub.
2. Import the project into Vercel.
3. The root directory is the current folder (`.`).
4. Vercel will automatically detect the build settings and deploy.
